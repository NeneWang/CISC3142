

# Markdown Reference

## Headings

```
# -> H1
## -> H2
### -> h3
```

## Styling

Style | syntax 
--|--
Bold | `**text**`
Italic | `*text*`
Strikethrough | `~text~`
Bold and Italic | `***text***
Quote | `> Quote`
Quoting | Use triple backstics
